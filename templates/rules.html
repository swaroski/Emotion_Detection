What is this platform ?


This platform provides a tool to interpret the facial emotions of the person watching a video. We have developed complex Deep Learning algorithm that analyze the results of the video and provides the client a summary of what our algorithm perceives, and how the video/advertisement performs .




Why this project ?


The project was developped as AI Deep Dive project. I have worked for several weeks on the individual models, and built this platform. I am using Python for the individual model training, Flask for the web interface, and D3.js for visualization purposes.
